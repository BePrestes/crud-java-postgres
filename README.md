**Crud Java + PostgreSQL**

Um sistema simples de cadastro de produtos e tipos, desenvolvido para fins de estudo e prática de backend.
Trabalhei arquitetura MVC, JDBC, PostgreSQL e boas práticas de desenvolvimento.

**Sobre mim**

Sou Bernardo Prestes, acadêmico de Engenharia de Software, apaixonado por tecnologia e focado em desenvolvimento backend.
Atualmente estudo Java, MySQL, PostgreSQL e Data Science, sempre buscando evoluir com projetos práticos.

**Funcionalidades**

> Cadastro de Produtos

> Cadastro de Tipos

> Relacionamento Produto ↔ Tipo

> CRUD completo (Create, Read, Update, Delete)

> Arquitetura MVC

> Conexão real com PostgreSQL (JDBC)

**Tecnologias Utilizadas*

> Java 17+	Backend / Lógica
> PostgreSQL	Banco de dados
> JDBC	Conexão com o banco
> Eclipse	IDE
> Git + GitHub	Versionamento
